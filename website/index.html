Terraform AWS Static Website Project with ☁️

This project demonstrates how to deploy a static website on AWS using Terraform, incorporating key components such as:

  • S3 Bucket: Hosts the static website files.
  • CloudFront Distribution: Delivers content globally with caching.
  • CloudWatch Monitoring: Monitors key metrics for AWS resources such as errors and request rates.
  • SNS Notifications: Sends alerts for critical events based on CloudWatch alarms.
  • Terraform Configuration: Sets up infrastructure as code for automation.

Explore our project to learn about the setup process, configuration files, and how you can replicate or extend this setup for your own projects.

Terraform Configuration

All infrastructure is managed as code using Terraform:

  • Define S3 bucket and CloudFront distribution configurations.
  • Establish IAM roles and policies for CloudWatch integration.
  • Automate deployment and updates to ensure consistency and scalability.
